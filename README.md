![Haunicoded House](asset/images/title.svg)

## 何これ

恐怖の屋敷、Haunicoded House へようこそ！
ここでは、おばけが文字を使って攻撃してきます。
あなたはその文字の Unicode のコードポイントを返してください。
制限時間はあなたのお好みのレベルから選ぶことができます。
さあ、あなたはいくつのおばけを退治することができるかな？
